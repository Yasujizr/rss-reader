          URL Rewriting

          Base URL: Required. The start of the URL. For example, in the url http://www.example.com/thengoto?nexturl=http://www.next-website.com/page.html, the base URL is http://www.example.com/thengoto (everything up to the ?). For example, in http://www.example.com/thengoto/http://www.next-website.com/page.html, the base URL is http://www.example.com/thengoto (everything up to the / after thengoto).
          Parameter: Optional. The name of the parameter. For example, in http://www.example.com/thengoto?nexturl=http://www.next-website.com/page.html, the parameter of interest is named nexturl. For example, in http://www.example.com/thengoto/http://www.next-website.com/page.html, there is no parameter of interest, so leave the Parameter field blank.
